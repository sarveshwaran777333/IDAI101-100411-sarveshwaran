NutriWell is a virtual nutrition coachHelps users:
1. Plan meals
2. Set fitness goals
3. Track progress
4. Stay motivated with reminders and tips.
Objective: Make personalized nutrition guidance accessible and friendly.

Many people struggle with healthy eating and exercise
Lack of personalized guidance leads to inconsistent habits.
NutriWell solves this by offering:
1. Personalized advice
2. Interactive reminders
3. Motivation

Dialogflow – For chatbot design
Custom Entities – diet_type, fitness_goal, meal_time, etc.
Rich Responses – Images, cards, quick replies, motivational messages

Flow:
1. User → Chatbot
2. NLP → Intent Matching
3. Response Generation → User

Intents and Their Purpose:
1. Greeting – Welcome user
2. Set Profile – Capture user details
3. Set Fitness Goal – Record user fitness goal
4. Hydration Reminder – Set water reminders
5. Exercise Recommendation – Suggest workouts
6. Daily Tips – Provide motivation
7. Track Meal – Record meals
8. Budget Meals – Suggest budget-friendly options
9. Goodbye – End conversation

Entities and Their Sample Values:
1. diet_type – Vegetarian, Vegan, Keto, Paleo
2. food_preference – Allergies, Likes, Dislikes
3. fitness_goal – Weight loss, Muscle gain, Maintain health
4. activity_level – Sedentary, Active, Very active
5. meal_time – Breakfast, Lunch, Dinner, Snack
6. budget – Low, Medium, High
7. cuisine – Indian, Italian, Chinese
8. time_available – 10 min, 20 min, 30 min
9. hydration_frequency – Every hour, 2 hours, 3 hours
10. reminder_type – Hydration, Exercise, Meal
